# Pin Mapping Table

| Function | ATmega32 Pin | Driver / Peripheral |
| --- | --- | --- |
| Start button | PB0 | GPIO input with pull-up |
| Stop button | PB1 | GPIO input with pull-up |
| Emergency button | PD2 | EXTI INT0 |
| Object sensor button | PD3 | EXTI INT1 |
| ADC speed input | PA0 | ADC channel 0 |
| Motor PWM speed pin | PB3 | PWM channel 0 |
| Motor direction IN1 | PD4 | GPIO output |
| Motor direction IN2 | PD5 | GPIO output |
| UART TX | PD1 | UART transmit |
| UART RX | PD0 | UART receive |
| Running LED | PC0 | GPIO output |
| Idle LED | PC1 | GPIO output |
| Fault LED | PC2 | GPIO output |
